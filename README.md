# University LMS UI

University LMS UI using HTML5 & CSS3 only.

## Description
A lightweight front-end prototype for a Learning Management System (LMS), built with semantic HTML and modular CSS. Intended for teaching UI structure and responsive design.

## Technologies
- HTML5
- CSS3

## Folder structure
lms-ui/

├── index.html
├── dashboard.html
├── courses.html
├── course-detail.html
├── assignments.html
├── profile.html

├── css/
│   ├── styles.css
│   └── components.css

└── images/

## How to push
1. Initialize git: `git init`
2. Add files: `git add .`
3. Commit: `git commit -m "Initial LMS UI structure"`
4. Add remote: `git remote add origin https://github.com/<username>/lms-ui-html-css.git`
5. Push: `git push -u origin main`
